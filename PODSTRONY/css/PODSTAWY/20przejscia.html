    Rodzaje timingu przejść:
    -ease (domyślny) - wolny start, potem szybko, wolny koniec
    -linear - prędkość bez zmian
    -ease-in - wolny start
    -ease-out - wolny koniec
    -ease-in-out - wolny start i koniec
    -cubic-bezier(n,n,n,n) - można zdefiniować samemu


    Składnia{
      transition: 1s color linear, 0.7s opacity ease-in;
    }
